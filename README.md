# Setup

## requirements

### Docker
https://www.docker.com/

### Make
Linux: `sudo apt install make`\
MacOS: `brew install make`\
WSL (Windows Subsystem for Linux): `sudo apt install make`

## Start the Webserver
```
cd /path/to/judentum-website/
sudo make run
```

## Stop the Webserver
```
cd /path/to/judentum-website/
sudo make stop
```
